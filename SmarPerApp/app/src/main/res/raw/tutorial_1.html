Below, the meaning of allow, obfuscate, and deny is explained for each private data category:

Contacts

  • Allow: Allows access to all contact info requested by the app.
  • Obfuscate: Allows access to contact names, phone numbers, email addresses, and postal addresses only:



  • Deny: Denies access and returns no contact data to the app.

Location

  • Allow: Allows access to your location.
  • Obfuscate: Returns an approximate location:



    The location of the center of the current square you are in gets returned.

  • Deny: Returns a fake location.

Storage

  • Allow: Allows access to your sdcard.
  • Obfuscate: Allows access to your sdcard, except for your public directories (Pictures, Music, Videos, etc):



    NOTE:     If you suspect an app needs write access to these directories (for example, to save a photo) you must allow Storage.

  • Deny: Denies all access to sdcard (i.e. sdcard unmounted).
