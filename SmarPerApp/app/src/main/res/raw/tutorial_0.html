SmarPer will prompt you upon app access to any of the following private resources (restriction categories in XPrivacy language):
  • Contacts
  • Location
  • Storage

You will see a prompt window as below:


Clicking on the ? reveals more information about decision options:


You must choose a semantic location before the Allow, Obfuscate, Deny buttons become available:


Cache: Your decision will be cached for 1 hour. Requests for the same resource/restriction category (i.e., Contacts, Location, Storage) made by the same app will have the same decision applied automatically.

Timeout will: Allow: If you fail to decide within the allotted time (40 seconds), Allow will be chosen. These timeout decisions are not added to the cache. Pressing ? will reset the 40-second timer and give you more time to respond.

Allow, Deny, Obfuscate: Your decision options. What each option does is explained in more detail on the next page.

A red prompt window: Request considered dangerous by XPrivacy, i.e. denying this request may cause the requesting app to crash.
